# NexaNews WordPress Theme

NexaNews is a modern, premium, fully responsive WordPress News/Magazine theme. It features a clean white design, sticky header with top bar, mega menu, hero news slider, latest news grid, trending sections, AJAX live search, custom login/register flows, and comprehensive Customizer options.

## Features
- Fully responsive modern design (mobile, tablet, desktop)
- Hero news slider on the front page
- Latest news grid and category sections
- Custom logo, primary navigation, and social menu support
- Built-in Customizer settings for colors, typography, and layouts
- SEO optimized with proper schema markup
- High performance (targets 90+ PageSpeed)

## Installation
1. Move the `/wp-content/themes/nexanews` directory to your WordPress `wp-content/themes/` directory.
2. Go to your WordPress Admin -> Appearance -> Themes.
3. Activate the "NexaNews" theme.

## Setup
To set up the homepage:
1. Create a new page titled "Home".
2. Create another page titled "Blog" or "News".
3. Go to Settings -> Reading.
4. Set "Your homepage displays" to "A static page".
5. Select "Home" as the Homepage and "Blog" as the Posts page.
6. Make sure to assign widgets to your Primary Sidebar in Appearance -> Widgets.
